# garlicts框架的开发计划

1. 集成zookeeper，实现分布式服务注册发现
2. 完善http请求工厂类，zookeeper + http工厂类 实现分布式http服务。
3. 集成分布式爬虫。
4. 集成MQ，多个消费者处理MQ的爬虫数据。
5. 集成分布式存储、实时计算。
6. 实现框架的组件化，可开关
7. 改造garlicts设计结构，使garlicts具备快速开发的能力

# Garlicts框架简介

分布式web框架，简单易用。

Garlicts框架的特色

1、使用更简单。
2、提供多方面的解决方案，解决问题更简单。
